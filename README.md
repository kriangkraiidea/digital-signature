# ✍️ Digital Signature · ลายเซ็นดิจิทัล

เว็บแอปสำหรับสร้างและบันทึกลายเซ็นดิจิทัล ใช้งานได้ทันทีผ่านเบราว์เซอร์

🔗 **เปิดใช้งาน:** https://kriangkraiidea.github.io/digital-signature/

## ฟีเจอร์

- ✏️ วาดลายเซ็นด้วยเมาส์หรือนิ้วบนจอสัมผัส
- 🖼️ อัปโหลดภาพลายเซ็นจากเครื่อง
- 💾 ดาวน์โหลดเป็น PNG โปร่งใส
- 📁 บันทึกลายเซ็นไว้ใน Drive (LocalStorage)
- 🎨 เลือกสีและขนาดปากกาได้

## วิธีใช้

1. กรอกชื่อ-นามสกุล
2. วาดลายเซ็น หรืออัปโหลดภาพ
3. กด **Save as PNG** เพื่อดาวน์โหลด
4. หรือกด **บันทึกลง Drive** เพื่อเก็บไว้ในเบราว์เซอร์

## เทคโนโลยี

- HTML5 Canvas
- Vanilla JavaScript
- LocalStorage API
